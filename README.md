# PolySaver
Vous souhaitez télécharger des vidéos de Youtube, Instagram, TikTok, Facebook dans le format et la qualité de votre choix ? PolySaver est fait pour vous. Codé en Rust, il permet un téléchargement rapide, efficace de jusqu'à 5 vidéos en parallèle. Il est également conçu pour être simple et intuitif. Téléchargez. Profitez.
